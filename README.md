# adidas-catalog-back
## Build with

- ReactJS
- styled-components
- Jest
- Enzyme
- Typescript
- Webpack
- Redux
- Axios
- esLint
- Storybook

## To Run
Follow this [link](https://github.com/linhphandk/adidas-catalog-compose) with backend.

## To run storybook
Storybook works without backend:
1. Run `npm install`
2. Run `npm run storybook`

## To run jest
Jest works without backend:
1. Run `npm install`
2. Run `npm run`
